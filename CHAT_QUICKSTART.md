# Chat Support System - Quick Start Guide

## For Users (Customers & Riders)

### Accessing Chat

1. **Log in** to your account
2. Click **"Chat Support"** in the left sidebar
3. Choose an existing conversation or click **"+ New"** to start a new chat

### How to Send a Message

1. Select a conversation from the left panel
2. Type your message in the input box at the bottom
3. Click **"Send"** or press Enter
4. Your message appears immediately with a timestamp

### Starting a New Chat

1. Click the **"+ New"** button
2. Select the user you want to chat with from the dropdown
3. Click **"Start Chat"**
4. Begin typing and send your message

### Viewing Conversation History

- All previous messages are shown in the conversation
- Messages are timestamped (e.g., "2:30 PM")
- Read/unread status shown with check marks
- Search conversations using the search box

## For Customers & Riders - Support Tickets

### Creating a Support Ticket

1. Click **"Support Tickets"** in the sidebar
2. Click the **"+ New"** button
3. Fill in the form:
   - **Subject**: Brief title of your issue
   - **Category**: Select the issue type
   - **Priority**: Choose Low, Medium, High, or Urgent
   - **Description**: Detailed explanation of the problem
4. Click **"Create Ticket"**

### Ticket Categories

| Category | Usage |
|----------|-------|
| **Delivery Issue** | Problems with order delivery |
| **Payment Problem** | Payment-related issues |
| **Order Issue** | Issues with order content/status |
| **Account Problem** | Login, profile, or account issues |
| **Refund Request** | Request for refund |
| **Rider Complaint** | Issues with rider behavior/service |
| **Bug Report** | App crashes or errors |
| **General Support** | Other inquiries |

## For Admin Staff

### Managing Conversations

1. Click **"Chat Support"** in sidebar
2. View all conversations with unread counts
3. Click a conversation to open
4. Answer customer/rider messages
5. Messages appear in real-time

### Handling Support Tickets

1. Click **"Support Tickets"** in sidebar
2. View all submitted tickets
3. Filter by status or search
4. Click a ticket to view full details
5. Add responses in the message area
6. Update ticket status when resolved

### Quick Actions

- **See unread messages** - Blue badges show unread counts
- **Search conversations** - Use the search box
- **View ticket details** - All ticket info at the top
- **Filter tickets** - Use tabs (All, Open, Resolved)

## Frequently Asked Questions

**Q: How long do messages stay in the chat?**  
A: Messages are stored permanently in your conversation history.

**Q: Can I send images or files?**  
A: Currently supports text messages. Image support coming soon.

**Q: Why isn't my message sending?**  
A: Check your internet connection and ensure you're logged in.

**Q: How do I close a support ticket?**  
A: Admin staff updates the status. You can reopen if needed.

**Q: Is my chat private?**  
A: Yes, conversations are only visible to the two participants (+ admins).

**Q: Can I delete a message?**  
A: Feature coming in next update.
